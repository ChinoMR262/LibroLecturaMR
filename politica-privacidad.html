I. Política de Privacidad Política de Privacidad del Sitio Web [ConfiAbleEmpress] En [ConfiAbleEmpress], la privacidad de nuestros usuarios es de suma importancia. Esta Política de Privacidad describe cómo recopilamos, utilizamos y protegemos su información personal cuando visita nuestro sitio web [URL del sitio web]. 1. No Recopilación de Información Personal Queremos dejar claro que [ConfiAbleEmpress] no recopila información personal de los usuarios. No solicitamos nombres, direcciones de correo electrónico, números de teléfono ni ningún otro dato que pueda identificarle. 2. Información Recopilada por Terceros Es posible que terceros, como proveedores de servicios de análisis web o redes publicitarias, recopilen información no personal sobre su visita a nuestro sitio web. Esta información puede incluir: Dirección IP: Una dirección IP es un número asignado a su dispositivo cuando se conecta a Internet. Tipo de navegador: El tipo de navegador que utiliza para acceder a nuestro sitio web (por ejemplo, Chrome, Firefox, Safari). Sistema operativo: El sistema operativo que utiliza en su dispositivo (por ejemplo, Windows, macOS, Android, iOS). Páginas visitadas: Las páginas que visita en nuestro sitio web. Tiempo de permanencia: El tiempo que pasa en cada página. Sitio web de referencia: El sitio web desde el que llegó a nuestro sitio web. Esta información se utiliza para analizar las tendencias, administrar el sitio web, rastrear el movimiento de los usuarios en el sitio web y recopilar información demográfica. Esta información no se utiliza para identificar a usuarios individuales. 3. Cookies Es posible que utilicemos cookies para mejorar su experiencia en nuestro sitio web. Las cookies son pequeños archivos de texto que se almacenan en su dispositivo cuando visita un sitio web. Puede deshabilitar las cookies en la configuración de su navegador, pero esto puede afectar la funcionalidad de algunas partes de nuestro sitio web. 4. Enlaces a Sitios Web de Terceros Nuestro sitio web puede contener enlaces a sitios web de terceros. No somos responsables de las prácticas de privacidad de estos sitios web. Le recomendamos que revise las políticas de privacidad de todos los sitios web que visite. 5. Seguridad Tomamos medidas razonables para proteger la información que recopilamos. Sin embargo, ninguna transmisión de datos a través de Internet o cualquier red inalámbrica puede garantizarse que sea 100% segura. Como resultado, si bien nos esforzamos por proteger su información, no podemos garantizar la seguridad de ninguna información que transmita a nosotros. 6. Cambios a esta Política de Privacidad Nos reservamos el derecho de modificar esta Política de Privacidad en cualquier momento. Le recomendamos que revise esta página periódicamente para estar al tanto de cualquier cambio. Los cambios entrarán en vigor inmediatamente después de su publicación en esta página. 7.
